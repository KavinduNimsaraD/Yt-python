# Yt-python

## Run

```bash
python GTA_BarChartRace_1080p.py
```

## Notes

- The script keeps a 1080p output (`16:9`, `dpi=120`) and targets ~380 seconds (6+ minutes).
- Milestone values are authored in **millions** and automatically scaled to **thousands** for finer-grained animation labels.
- Output video file: `GTA_Evolution_1080p.mp4`.
