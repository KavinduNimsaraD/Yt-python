# Yt-python

Generate a 1080p GTA sales bar chart race video using **Ten-Thousands units**.

## Run

```bash
pip install pandas numpy matplotlib bar_chart_race
python GTA_BarChartRace_1080p.py
```

## Output

- File: `GTA_Evolution_1080p.mp4`
- Resolution: 1920×1080 (16:9)
- Timeline: Monthly (`MS`) with smooth interpolation
- Display Unit: **Ten-Thousands (×10,000 copies)**
  - Example: value `21500` means `215,000,000` copies
